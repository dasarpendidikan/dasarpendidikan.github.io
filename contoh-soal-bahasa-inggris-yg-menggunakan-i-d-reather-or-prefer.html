19. 2. di you prefer dancing or singing l prefer to ​


║STATING PREFERENCE║

[tex]Answer[/tex]

Do you prefer dancing or singing? I prefer singing to dancing.

[tex]Explanation[/tex]

PREFERENCEUngkapan perbandingan perasaan lebih menyukai terhadap sesuatu/seseorang.Tiap preference memiliki konjungsi masing-masing yang menjadi pasangannya. Konjungsi ini sebagai perbandingan terhadap sesuatu/seseorang.

KINDS OF PREFERENCE :

Penggunaan "prefer"  ada 2 jenis yaitu dengan diikuti Gerund { the word "prefer" is one of common verbs should be followed by verb-ing  } dan TO-infinitive .

1. By using PREFER .........TO

PATTERN  :

S+prefer (s)+Gerund/noun +TO+Gerund/noun

Example :

1. I prefer writing to speaking ( by using Verb-ing )

{ saya lebih suka menulis daripada berbicara }

2. They prefer car to bus ( by using noun )

{ Mereka lebih suka mobil dari pada bis }

2. By using WOULD PREFER TO .............RATHER THAN

Penggunaan "prefer" dengan to-infinitive. " would prefer " bisa di kontraksi menjadi "  'd prefer ."

PATTERN :

(+) S+would prefer+to+verb 1+object+ rather than+verb1+object

(-) S+would prefer+not+to+verb1+object+rather than+to+verb 1+object

(?) Would+S+prefer+to+verb 1+object+rather than+to+verb1+object?

Example :

(+) He would prefer to eat fruit salad rather than (to) eat fried chicken.

(-)  He would prefer not to eat fruit salad rather than (to) eat fried chicken

(?) Would he prefer to eat fruit salad rather than (to) eat fried chicken? or we can say would he prefer to eat fruit salad or fried chicken?

3. By using  WOULD RATHER..............THAN

PATTERN :

(+) S+Would rather+verb1 + Object + than + verb1 + object

(-) S + would rather + not + verb 1 + object + than + verb 1 + object

(?) Would + subject + rather + verb1 + object + than + verb 1 + object?

Example :

(+) He would rather read some books than play outside.

(-)  He would rather not read some books than play outside

(?) Would he rather read some books than play outside? or we can say Would he rather read some books or play outside?

4. By using LIKE .............BETTER THAN

PATTERN :

Subject +like(s)+ noun/gerund+better than+noun/gerund.

Example :

1. I like eating the fruits better than (eating) the meat   ( by using verb-ing )

{ Aku lebih suka makan buah-buahan daripada daging }

2. I like an English grammar book better than a history book (by using noun/noun phrase)

{ Aku lebih suka buku grammar bahasa inggris daripada buku sejarah }

36. mohon bantuannya kakapa yg dimaksud dengan prefer/prefersapa artinya kedua kata tersebutdan bagaimana cara menggunakan dua kata tersebut, contoh kalimat dan dialog nya dan apa bedanya prefer/prefers​


Jawaban:

Kata "Prefer" digunakan dalam kalimat untuk menyatakan Preferensi atau lebih suka akan sesuatu atau lebih suka untuk melakukan sesuatu. Dalam belajar bahasa Inggris dasar, prefer bisa diartikan lebih suka dari.

Penggunaan Kata Prefer dan Prefers

Kata "prefer" digunakan untuk subjek I, they, we, dan you. Sedangkan kata "prefers" digunakan untuk subjek he, she, dan it.

Berikut adalah Beberapa Contoh Kalimat yang menggunakan Kata Prefer

- I Prefer to go to the beach than go to the zoo

- We Prefer going by Plane

- We have tea and coffee, but perhaps you’d prefer a cold drink.

Berikut adalah Beberapa Contoh Kalimat yang menggunakan Kata Prefers

- She Prefers that we go to the Park

- He Prefers to go with his sister

Berikut adalah Contoh Dialog menggunakan Kata Prefer

Julie : Hey Kate! What are you doing here?

Kate : Hi Julie! I'm here because My Little sister wants to Buy ice cream

Julie : I see, But the Ice Cream shop is close now.

Kate : Oh right! What should I do?

Julie : Maybe, You can make new choices for your Little Sister!

Kate : That's a good idea!

Lucy ( Young Sister ) : Sister, Where is my Ice cream?

Kate : I'm sorry Lucy, The Ice Cream shop is close! Maybe we can go to another places. I have 2 Choices for you, Which one do you prefer? go to the Candy shop or Toys store?

Lucy ( young sister ) : I Prefer to go to the Candy shop!

Kate : Alright then, Let's go!

Berikut adalah Contoh Dialog menggunakan Kata Prefers

Jennifer : Hey Yuki! Come here!

Yuki : Oh hey Jennifer, Sorry but I can't. cause I'm waiting for Anna here.

Jennifer : Oh Alright then. What will you do with Her?

Yuki : We will Go to the Park together. will you join us?

Jennifer : Sure! I'd love to!

10 Minutes later

Jennifer : Where is she, Yuki?

Yuki : I don't know. Maybe I can chat her!

Jennifer : Okay.

Yuki : Umm, She said that we can't go to the Park! cause the Park is Crowded. So, She Prefers to go to the Restaurant. What do you think Jennifer?

Jennifer : I Don't mind to go to the restaurant.

Yuki : Okay Then, let's Go..
