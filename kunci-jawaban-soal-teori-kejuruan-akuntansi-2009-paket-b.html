15. Dalam satu paket soal ujian terdiri dari paket A sebanyak 5 soal dan paket B sebanyak 6 soal.banyak cara menjawab 5 soal,jika 3 soal dari paket A dan 2 soal dari paket B adalah...cara


Menggunakan kombinasi berdasarkan variasi soal adalah sama apabila berbeda urutan.
Berlaku:
n = Soal A x Soal B
n = C(5,3) x C(6,2)
n = 5!/(2!3!) x 6!/(4!2!)
n = 5.4/2 x 6.5/2
n = 10 x 15
n = 150 cara

16. kunci jawaban buku paket matematika hal 214 buku paket. (yg nomor genap saja)


Pembahasan

Operasi penjumlahan dan pengurangan pada bentuk aljabar dapat dilakukan dengan ketentuan, yaitu :

pada suku-suku sejenis.mengumpulkan suku-suku sejenis.

Suku-suku sejenis dapat diartikan sebagai suku yang memiliki variabel dan pangkat yang sama. variabel biasanya dilambangkan sebagai a, b, c, ...., z

Balik Pada Soal :

3] Tentukan hasil pengurangan bentuk aljabar berikut

a) (42n + 35m + 7) − (−50m − 20n + 9)

    Penyelesaian :

    = (42n + 35m + 7) − (−50m − 20n + 9) → perkalian tanda

    = 42n + 35m + 7 + 50m + 20n - 9 → gabungkan suku sejenis

    = 42n + 20n + 35m + 50m + 7 - 9 → Jumlahkan suku sejenis

    = 62n + 85m - 2


b] (5x + 3) − (x − 1) =

   Penyelesaian :

   = (5x + 3) − (x − 1) → Perkalian tanda

   = 5x + 3 - x + 1      → gabungkan suku sejenis

   = 5x - x + 3 + 1      → Jumlahkan suku sejenis

   = 4x + 4


c] (2y + 15z) − (4y − 8)

   Penyelesaian :

   = (2y + 15z) − (4y − 8) → Perkalian tanda

   = 2y + 15z - 4y + 8      → gabungkan suku sejenis

   = 2y - 4y + 15z - 8       → jumlahkan suku sejenis

   = -2y + 15z - 8

---------------------------------------------------------------------------------------------------

4] Tentukan Hasil pengurangan 5z + 3 oleh 2z - 7

Penyelesaian :

 = (5z + 3) - (2z - 7) → Perkalian tanda

 = 5z + 3 - 2z + 7     → gabungkan suku sejenis

 = 5z - 2z + 3 + 7     → jumlahkan suku sejenis

 = 3z + 10

---------------------------------------------------------------------------------------------------

5] Tentukan hasil pengurangan 6x + 4 oleh x - y

Penyelesaian :

= (6x + 4) - (x - y) → perkalian tanda

= 6x + 4 - x + y     → gabungkan suku sejenis

= 6x - x + y + 4     → jumlahkan suku sejenis

= 5x + y + 4

---------------------------------------------------------------------------------------------------

6] Kurangkalah :

a] 5x - 9 dengan 7x + 15y

   Penyelesaian :

   = (5x - 9) - (7x + 15y) → Perkalian tanda

   = 5x - 9 - 7x - 15y      → gabungkan suku sejenis

   = 5x - 7x - 15y - 9

   = -2x - 15y - 9


b] 5x - 3y + 7 dari 5y − 3x − 4

   Penyelesaian :

   = (5y - 3x - 4) - (5x - 3y + 7)  → Perkalian tanda

   = 5y - 3x - 4 - 5x + 3y - 7    → gabungkan suku sejenis

   = 5y + 3y - 3x - 5x - 4 - 7    → jumlahkan suku sejenis

   = 8y - 8x - 11

panjang sisi terpanjang (3x + 6) cm. Jika panjang sisi sisanya (x + 6),

maka tentukan keliling segitiga tersebut

Penyelesaian :

sisi pendek = (2x - 5) cm

sisi panjang = (3x + 6) cm

 sisi lainnya = (x + 6) cm


Maka, Keliling segitiga = sisi + sisi + sisi

                                      = (2x - 5) + (3x + 6) + (x + 6)

                                      = 2x + 3x + x - 5 + 6 + 6

A. jawablah pertanyaan-pertanyaan berikut berdasarkan kutipan teks berikut

1. Apa yang dimaksud dengan teks persuasi?

Jawaban:

Teks persuasi adalah teks yang berusaha meyakinkan pembaca sehingga berkenan melakukan sesuatu yang diinginkan oleh si penulis teks.

Teks Persuasi ini juga bisa diartikan sebagai teks yang tujuannya adalah menyuruh, membujuk, mengajak dan semacamnya berbuat sesuatu sesuai kehendak sang penulis

2. Hal apa saja yang membedakan teks persuasi dengan jenis teks lainnya?

Jawaban:

Hal-hal yang membedakan teks persuasi dengan teks yang lain yakni:

- Teks persuasi berisi pernyataan-pernyataan yang berupa ajakan atau bujukan, yang bertujuan mendorong seseorang untuk mengikuti harapan atau keinginan-keinginan penulisnya.

- Pada teks persuasi terdapat pendapat atau fakta yang digunakan untuk memengaruhi pembaca agar mau mengikuti bujukan-bujukan.

3. Apa yang diharapkan penulis teks persuasi untuk pembacanya?

Jawaban:

Yang diharapkan penulis teks teks persuasi untuk pembacanya adalah pembaca berbuat sesuatu sesuai dengan harapan penulisnya.

Sementara yang diharapkan penulis teks persuasi untuk pembacanya dari kutipan teks di atas ialah pembaca dapat membagi waktunya antara kegiatan nonakademik dengan belajar

4. Apa fungsi fakta di dalam teks persuasi?

Jawaban:

Fungsi fakta dalam teks persuasi adalah untuk menguatkan argumen-argumen yang disampaikan oleh penulis atau pembicara.

5. Mengapa cuplikan teks tersebut persuasif? Jelaskan!

Jawaban:

Cuplikan teks termasuk persuasif karena isi dari teks tersebut adalah ajakan untuk para pembaca mengikuti opini dari penulis.
