18. Choose one underlined word or phrase which is incorrect.

Pelajaran: English
Kategori: grammar review
Pembahasan: 

The way I see it, there are mistakes here and there in this sentence. As for the incorrect one from the options, the primary mistake is A (as it should be 'having'), the secondary mistakes are in B (which should be without 'having' as it's not necessary) and C (which should be without 'they are' with the same reason as above plus the wrong spelling, it should be 'unreliable.'). Finally, the tertiary mistake is the word 'little', it should be using the word 'few' as 'friend' is countable. Thus, the final correct sentence should be 
'I prefer having a few but really good friends than many friends but unreliable.'
