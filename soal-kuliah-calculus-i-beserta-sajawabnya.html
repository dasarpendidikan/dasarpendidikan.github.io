1. [tex](\frac{1}{1+y})cos(\frac{x}{1+y})[/tex]

Penjelasan:

Karena ini turunan dengan variabel x, maka y menjadi konstan, sehingga kita bisa menurunkannya, yaitu dengan cara menurunkan "sin(t) dx". Kita tahu bahwa turunan dari sin(t) dengan variabel x adalah "cos(t) t' dx"

kenapa adalah t'? Karena kita harus menggunakan "Chain Rule", yaitu dengan mengalikan dengan derivatif dari sesuatu yang kita substitusikan (t = x/(1+y), turunannya adalah 1/(1+y))

Maka hasil akhir:

[tex]\frac{df}{dx} = \frac{d}{dx}{sin(\frac{x}{1+y})}\\\\cos(\frac{x}{1+y})(\frac{1}{1+y})\\\\(\frac{1}{1+y})cos(\frac{x}{1+y})[/tex]

Soal No. 1

Berapakah hasil pemfaktoran dari bilangan berikut ini 16x2 − 9y2 ?

Jawabannya :

Harus di ingat bahwa bentuk faktor nya yakni aljabar :

a2 – b2 = ( a + b ) ( a – b )

16x2 = ( 4x )2

9y2 = ( 3y )2

Sehingga faktor dari bilangan 4x2 – 9y2 ialah ?

16x2 – 9y2 = ( 4x + 3y ) ( 4x – 3y )

Jadi, hasil pemfaktoran dari bilangan 16x2 − 9y2 ialah : ( 4x + 3y ) ( 4x – 3y ).

33. Untuk menerangkan karakteristik mata kuliah X, misalkan p: "Kuliahnya menarik", dan q: "Dosennya enak", r: "Soal-soal ujiannya mudah". Terjemahkan proposisi-proposisi berikut dalam notasi simbolik (menggunakan p, q, r): (a) Kuliahnya tidak menarik, dosennya tidak enak, dan soal-soal ujiannya tidak mudah. (b) Kuliahnya menarik atau soal-soal ujiannya tidak mudah, namun tidak keduanya. (c) Salah bahwa kuliahnya menarik berarti dosennya enak dan soal-soal ujiannya mudah.​


Jawaban:

Dalam notasi simbolik menggunakan p, q, dan r, kita dapat menerjemahkan proposisi-proposisi berikut:

(a) Kuliahnya tidak menarik, dosennya tidak enak, dan soal-soal ujiannya tidak mudah.

Notasi simbolik: ¬p ∧ ¬q ∧ ¬r

(b) Kuliahnya menarik atau soal-soal ujiannya tidak mudah, namun tidak keduanya.

Notasi simbolik: (p ∨ ¬r) ∧ ¬(p ∧ ¬r)

(c) Salah bahwa kuliahnya menarik berarti dosennya enak dan soal-soal ujiannya mudah.

Notasi simbolik: ¬(p → (q ∧ r))

Penjelasan:

- Simbol ¬ digunakan untuk menyatakan negasi (tidak).

- Simbol ∧ digunakan untuk menyatakan konjungsi (dan).

- Simbol ∨ digunakan untuk menyatakan disjungsi (atau).

- Simbol → digunakan untuk menyatakan implikasi (jika...maka).

Jawab:

1.  aljabar linear = 600 mahasiswa

2.  Bahasa pemograman =  300 mahasiswa

3.  sistem operasi I =  200 mahasiswa

Penjelasan dengan langkah-langkah:

jika ; Aljabar Linear = a

       bahasa pemograman = b

dan sistem operasi I = c

jika nilai a, b dan c selalu sama, maka dalam skala 1% didapat :

10a + 20b = 120 mahasiswa

6a + 15c = 60 mahasiswa

30b + 20c = 145 mahasiswa

⇔ {(10a + 20b = 120) x 3}  -  {(30b + 20c = 145) x 2}

⇔ (30a + 60b = 360)  -  (40c + 60b = 290)  =  30a - 40c = 70

⇒⇒ {(6a + 15c = 60) x 5}  -  (30a - 40c = 70)

⇒⇒ (30a + 75c = 300)   -   (30a - 40c = 70)   ⇔  115c = 230

⇒ 115c = 230  ⇒ c = 230/115 = 2

⇒⇒⇒ 30b + 20c = 145  ⇒ 30b + (20x2) = 145  ⇒  30b + 40 = 145  ⇒ 30b = 145 - 40  

⇒ 30b = 105  ⇒ b = 105/30  ⇒  b = 3

⇒ 10a + 20b = 120

⇒ 10a + (20 x 3) = 120

⇒ 10a + 60 = 120

⇒10a = 120 - 60

⇒10a = 60

⇒ a = 60/10 = 6

didapat nilai a = 6, b = 3 dan c = 2

maka jumlah masing-masing pada mata kuliah adalah:

1.  aljabar linear = a x 100 = 6 x 100 = 600 mahasiswa

2.  Bahasa pemograman = b x 100 = 3 x 100 = 300 mahasiswa
