<=> 32 = 1/2 x a²
<=> a² = 64
<=> a  = √64
<=> a  = 8

a = t = 8

Panjang sisi miring:
s² = a² + t²
s² = 8² + 8²
s² = 2 x 8²
s  = √(2x8²)
s  = 8√2 cm
