∫ k f(x) dx = k ∫ f(x) dx

[tex]\int {\frac{1}{2} sin \ 9x } \, d(9x)+\int {\frac{1}{2} sin \ x } \, dx\\ \\ = \frac{1}{2} \int {sin \ 9x } \, d(9x)+\frac{1}{2} \int {sin \ x } \, dx\\ \\ = -\frac{1}{2}(cos \ 9x+ cos \ x)[/tex]
