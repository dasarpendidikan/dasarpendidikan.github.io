29. no gasal no bahasa elien​


Jawaban

72⁰

Penyelesaian

Satu garis lurus = 180⁰

8x - 20 + 4x + 8 = 180

8x + 4x = 180 + 20 - 8

12x = 192

[tex]x = \frac{192}{12} [/tex]

x = 16

Ditanya : Sudut pelurus AOC, maka yang ditanya adalah sudut BOC

BOC = 4x + 8

BOC = 4(16) + 8

BOC = 64 + 8

BOC = 72⁰

Jawaban:

d. 72°

Penjelasan :

Dua sudut dikatakan berpelurus jika besar sudutnya 180°

∠AOC + ∠BOC = 180°

8x - 20 + 4x + 8 = 180°

8x + 4x - 20 + 8 = 180°

12x - 12 = 180°

12x = 180° + 12°

12x = 192°

x = 192/12

x = 16°

karena yang ditanya sudut pelurus nya berarti yang di cari sudut disebelahnya

Pelurus ∠AOC adalah ∠BOC

∠AOC = ∠BOC

∠BOC = 4x + 8

= 4(16°) + 8°

= 64° + 8°

= 72° (D)

Berarti jawabannya adalah D. 72°
