Pertanyaan :
Bagaimana cara instal ulang laptop?

Jawab :
Untuk menginstal ulang laptop kamu harus tahu dasar sistem operasi dan hardware. Ga perlu banyak-banyak. Cuma dasarnya saja seperti driver, hardware laptop. Kalau kamu ga tahu dasar sistem operasi, akan menyebabkan masalah nantinya.

Untuk menginstal ulang laptop kamu perlu :
1. Perangkat bootable (flashdisk atau CD) berisi file Windows / sistem operasi lain (kalau mau)
2. Laptop dan speknya

Langkah-langkah instal ulang laptop Windows 10 :
1. Hidupkan laptop, masukkan perangkat untuk booting, masuk bios, dan pilih "1st boot" => Flashdisk / CD
2. Ketika ada tulisan "press any key for continue", tekan sembarang tombol
3. Ketika sudah dimenu instal, pilih bahasa instal lalu klik "install now"
4. Tunggu hingga muncul EULA. Baca dulu persetujuannya dan lanjutkan dengan menekan "I Agree"
5. Pilih tipe instalasinya (pilih install Windows only". Masukkan product keynya. Untuk Windows 10 bisa lanjut tanpa dengan Product key dengan menekan "I don't have product key"
6. Pilih partisi untuk diinstal sistem operasinya lalu klik next. Tunggu hingga instalasi selesai.
7. Ketika sudah komputer akan restart. Keluarkan cd instalasinya
8. Lanjutkan setup laptopnya dari nama komputer, sinkronisasi, dll
9. Komputer sudah selesai diinstal. Lebih baiknya kamu instal driver tambahan buat hardwarenya. kalo enggak sistemnya terganggu. gunakan CD laptopbyang ada untuk menginstal driver karena kompatibel dengan hardware aslinya

CARA MENGINSTALL SOFTWARE DI KOMPUTER / LAPTOP

Bagi anda pengguna PC/Laptop pastinya tidak asing lagi dengan kata Menginstal/Instal software.Yap, Install adalah proses pemasangan software tertentu kedalam sistem komputer/laptop contohnya game dan antivirus,sebelum bisa digunakan pastinya akan melalui proses penginstallan terlebih dahulu.
Nah sesuai judul artikel kali ini, dibawah ini adalah salah satu contoh proses penginstallan sebuah software beserta penjelasannya,jadi harap disimak baik-baik ya.

1.Pertama kita siapkan file installer software yang ingin kita install,kemudian double klik pada file installer tersebut,setelah kita klik file instaler softwarenya biasanya muncul pesan "Do You Want To Run This File",klik run jika ingin melanjutkan ke penginstalan 


2.Setelah anda mengklik run,biasanya anda akan masuk ke menu utama software tersebut,jika anda ingin menginstall software tersebut klik next

￼

3.Kemudian,jika muncul license seperti ini, berarti anda diminta untuk menyetujui license tersebut, untuk melanjutkan pilih "I Accept / I Agree" kemudian klik next

￼


4.Ini adalah bagian untuk pemilihan lokasi dimana software tersebut akan diinstall/disimpan, biasanya secara default akan tersimpan di "C:/Program files" namun jika anda ingin menyimpan/menginstall software tersebut di tempat/drive lain anda dapat mengklik browse,tetapi jika anda ingin menginstall software tersebut di lokasi default,anda dapat langsung mengklik Next

￼


5.Setelah memilih lokasi file yang diinstall biasanya proses pemecahan/penginstalan file akan dimulai,tunggu hingga selesai dan biasanya jika sudah selesai akan muncul pesan installation complete/penginstalan selesai, kemudian klik finish untuk menyelesaikan penginstallan.


Selesai,semoga cara menginstall software di komputer/laptop ini dapat bermanfaat dan jika ada yang kurang jelas bisa tanyakan lewat komentar,terima kasih.

35. bagaimana cara menginstal leptop


1. Nyalakan laptop, kemudian masuk ke menu bios dengan cara menekan tombol 'del' saat laptop mulai menyala.

2. Setelah masuk ke menu bios, pilihlah menu boot. Di sini kita harus memilih boot device dengan urutan pertama adalah CD/DVD ROM, dan save setting bios. Jadi pada saat laptop dinyalakan, otomatis akan membaca CD/DVD ROM yang sudah dimasukkan CD Windows, dan kemudian akan berlanjut cara instal laptop tahap selanjutnya ini.

3. Masukkan CD Windows ke dalam CD/DVD Rom, kemudian restart laptop.

4. Ketika laptop baru menyala, tunggu beberapa saat dan akan muncul tulisan "press any key bla bla bla..." tekan saja sembarang tombol di keyboard, (enter misalnya).

5. Proses loading... kemudian akan muncul tiga pilihan, enter saja untuk memilih To set up Windows XP now, press ENTER.

6. Kemudian akan muncul halaman license agreement, tekan saja tombol F8.

7. Kita asumsikan bahwa hardisk laptop sudah terpartisi biasanya dan seharusnya ada dua partisi, menjadi partisi C dan partisi D. Bila belum baca dulu: cara membuat partisi hardisk baru (cooming soon...).

8. Pilih pada partisi C dengan menggunakan tombol anak panah naik turun pada keywboard.

9. Selanjutnya akan muncul pilihan untuk melakukan format hardisk, pilih saja "format the partision using the NTFS file system, enter dan tunggu loading.

10. Sampai tahap cara instal laptop baru ini, kita tinggal menunggu saja. Akan ada proses format hardisk pada partisi C sampai proses selesai 100 persen, kemudian otomatis dilanjutkan copy file windows dari CD Windows, tunggu sampai proses copy selesai.

11. Saat proses copy file windows ke hardisk selesai, laptop akan otomatis restart, untuk mempercepat nya tekan saja tombol ENTER.

12. Setelah laptop restart, akan muncul kembali pilihan seperti tahap nomor 4, jangan tekan tombol apapun, biarkan saja dan otomatis proses instal laptop akan berjalan, dan kita hanya suruh menunggu sampai tahap selanjutnya.

13. Saat proses copy file Windows sudah selesai, tunggu sampai ada perintah "Regional and Language Option", pilih saja NEXT.

14. Selanjutnya ada pilihan untuk mengatur waktu (jam), pilih di menu dropdown nya : (GMT+7:00 Bangkok, Hanoi Jakarta) dan tekan NEXT.

15. Pada tahap cara instal laptop baru selanjutnya adalah, kita harus memasukkan Product Key, atau biasa kita sebut dengan serial number Windows, masukkan dan tekan NEXT.

16. Tahap selanjutnya adalah memasukkan nama kamu dan nama organisasi kamu, tulis saja terserah dan NEXT.

17. Tahap berikutnya mengisi computer name dan administrator password, abaikan saja atau isi tetapi harus diingat-ingat administrator password nya, NEXT.

18. Kemudian masuk ke menu Networking setting, biarkan saja seperti bawaan aslinya, tekan NEXT.

19. Selanjutnya muncul menu Workgroup or Computer Domain, biarkan saja seperti default Windows, tekan NEXT.

20. Selanjutnya kita tinggal menunggu saja, proses sedang berjalan.

21. Sampai muncul tulisan Display Setting "To improve the appereance... bla bla bla..." tekan saja tombol OK.

22. Muncul lagi tulisan Monitor Setting "Windows adjusted your screen... dst..." tekan tombol OK lagi.

23. Tunggu beberapa saat, dan tratatataa... Windows kita sudah siap sekarang dan proses cara instal laptop baru kita sudah selesai, dan kita bisa untuk memulai instal program lain yang diperlukan.
