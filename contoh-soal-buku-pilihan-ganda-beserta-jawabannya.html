1. A = {5,6,7}

B = {3,4}

Maka nilai dari A ∪ B = {3,4,5,6,7}

2. M = {a, i, u, e, o}

N = {a, u, o}

M ∪ N = {a, i, u, e, o}

Nilai n (M ≦ N) = 5

Jawabannya: A

3. X = {x | x <6, x bilangan angka asli)

= {1, 2, 3, 4, 5}

Y = {x | – 1 ≤ x ≤ 5, x є nilai integer}

= {-1, 0, 1, 2, 3, 4, 5}

(X ∩ Y) = {1, 2, 3, 4, 5}

Jawabannya: B

4. n (A) = 10, n (B) = 8 serta n (A ≤ B) = 8

n (A – B) = n (A) + n (B) -n (A-B)

= 10 + 8 – 8

= 10

Jawabannya: C
