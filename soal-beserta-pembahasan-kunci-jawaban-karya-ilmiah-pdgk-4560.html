hasil dari [tex]4.560 : (-15) × (-123)[/tex] adalah [tex]\sf{\red{37.392}}\: \sf{\underline{\green{(Opsi \: D)}}}[/tex]

____________Pendahuluan

Operasi hitung bilangan bulat adalah menghitung jumlah angka supaya diketahui nilai hasilnya. Dalam operasi bilangan perkalian dan pembagian biasanya terdapat bilangan positif dan negatif. Dalam menghitung bilangan positif dan negatif dalam operasi perkalian dan pembagian, kalian bisa melihat tabel dibawah ini :

1. Perkalian

[tex]\begin{gathered}\boxed { \begin {array} {clclc} {\underline{\bold{ perkalian}}{} } \\ \\ \sf( + ) \times ( + ) = ( + ) \\ \sf( + ) \times ( - ) = ( - ) \\ \sf( - ) \times ( - ) = ( + ) \\ \sf( - ) \times ( + ) = ( - ) \end{array}}\end{gathered}[/tex]

dapat kita simpulkan bahwa :

[tex] \sf{\underbrace{setiap \: berbeda \: maka}} \\ - {\sf(negatif)}[/tex][tex]\sf{\bold{\underbrace{setiap \: yang \: sama \: maka}}} \\ + \: \sf{(positif)}[/tex]

2. Pembagian

[tex]\begin{gathered}\boxed { \begin {array} {clclc} {\underline{ \bold{ pembagian}}{} } \\ \\ \sf( + ) \div ( + ) = ( + ) \\ \sf( + ) \div ( - ) = ( - ) \\ \sf( - ) \div ( - ) = ( + ) \\ \sf( - ) \div ( + ) = ( - ) \end{array}}\end{gathered}[/tex]

dapat kita simpulkan, bahwa :

[tex] \sf{\bold{\underbrace{setiap \: yang \: sama \: hasilnya}}} \\ \sf{\bold{\underline{ + \: (positif)}}}[/tex][tex]\bf{\bold{\underbrace{setiap \: yang \: beda \: hasilnya}}} \\ \bf{\bold{\underline{ - \: (negatif)}}}[/tex]_____________Pembahasan

Diket

Hasil 4560 : (-15) x (-123) = ….

Dit

Selesaikan operasi tersebut !

Jawab

[tex](4.560 \div ( - 15)) \times ( - 123)[/tex][tex] - 304 \times ( - 123) [/tex][tex] \boxed{\sf{\orange{37.392}}}[/tex]____________Penjelasan

[tex]\sf{\underbrace{4.560 \div ( - 15) = - 304}} \\ \sf{\bold{hasilnya \: negatif \: sebab \: ( + ) \div ( - ) \: hasilnya \: {\red{negatif}}}}[/tex][tex]\sf{\underbrace{ - 304 \times ( - 123) = 37.392}} \\ \sf{\bold{hasilnya \: positif \: sebab \: ( - ) \times ( - ) \: hasilnya \: {\red{\underline{positif}}}}}[/tex]____________Kesimpulan

Jadi hasil dari [tex]4.560 \: ÷ \: (-15) \times (-123)[/tex] adalah [tex]\sf{\red{37.392}}[/tex]
