5. contoh soal tentang barisan bilangan matematika beserta dengan jawabannya​


Contoh Ke 1

Diketahui rumus ke – n suatu barisan adalah Un = 10n + 3. Hasil penjumlahan nilai suku ke – 22 dan ke – 24 adalah ….

A. 482

B. 466

C. 470

D. 482

Pembahasan:

Rumus suku ke – n: Un = 10n + 3

Mencari nilai suku ke – 22:

U22 =10 x 22 + 3

U22 = 220 + 3

U22 = 223

Mencari nilai suku ke – 24:

U24 =10 x 24 + 3

U24 = 240 + 3

U24 = 243

Hasil penjumlahan nilai suku ke – 22 dan ke – 24:

U22 + U24 = 223 + 243 = 466

Jawaban: B

Contoh Ke 2

Diketahui barisan bilangan: 3, 8, 13, 18, 23, …

Suku ke-32 adalah ….

A.     465

B.     168

C.     158

D.     153

Pembahasan:

Berdasarkan pola bilangan pada soal, dapat diperoleh bawah suku pertamanya adalah 3 (a = 3) dan beda setiap sukunya adalah 5 (b = 5).

Un = a + (n – 1)b

U32 = a + 31b U32 = 3 + 31 × 5

U32 = 3 + 155 = 158

Jawaban: C

 

Contoh Ke 3

Diketahui barisan bilangan: −3, 1, 5, 9, 13, …

Suku ke – 52 adalah …

A.     201

B.     207

C.     208

D.     215

Pembahasan:

Berdasarkan pola bilangan pada soal, dapat diperoleh bawah suku pertamanya adalah -3 (a = -3) dan beda setiap sukunya adalah 4 (b = 4).

Un = a + (n – 1)b

U52 = a + 51b

U52 = – 3 + 51 × 4

U52 = – 3 + 204 = 201

Jawaban: A

9. SOAL PAS MATEMATIKA SE 1. Diketahui pola bilangan 17, 14, 11, 8.... Suku ke -7 dari pola bilangan tersebut adalah......​


Jawaban:

-1

Penjelasan dengan langkah-langkah:

Untuk mencari suku ke-n menggunakan rumus Un = a + ( n - 1 ) × b

Keterangan:

n = suku yang ingin dicari

a = nilai suku pertama

b = beda setiap suku

Yang diketahui:

a = 17

b = 14 - 17 = -3

n = suku ke-7

Langkah Langkah Pengerjaan:

U7 = 17 + ( 7 - 1 ) × (-3)

U7 = 17 + 6 × (-3)

U7 = 17 + (-18)

U7 = -1

Ada 4 macam pola bilangan

Pola bilangan persegi panjang [ [tex]U_n = n(n+1)[/tex] ]Pola bilangan persegi [ [tex]U_n = n^2[/tex] ]Pola bilangan segitiga [ [tex]U_n = \frac{n}{2} (n+1)[/tex] ]Pola bilangan pascal [ [tex]U_n = 2^{n-1}[/tex] ]

Rumus barisan aritmatika

[tex]U_n = U_1 + (n-1)b[/tex]

[tex]S_n = \frac{n}{2} (a+U_n)[/tex] atau [tex]S_n = \frac{n}{2}(2a + (n-1)b)[/tex]

[tex]U_n[/tex] = Suku ke-n

[tex]S_n[/tex] = Jumlah n suku pertama

n = banyak suku barisan aritmatika

b = beda

a = suku pertama barisan aritmatika

PEMBAHASAN

1. Dik : 1, 4, 9, 16, 25, ...

Dit : Suku ke 20?

Jawab : [tex]1 = 1^2, 4 = 2^2, 9 = 3^2, 16 = 4^2, ...[/tex]

[tex]U_n = n^2\\U_{20} = 20^2 = 400[/tex]

Jadi, suku ke 20 dari pola bilangan 1, 4, 9, 16, 25, ... adalah 400

2. Dik : 2, 6, 12, 20, ...

Dit : Suku ke 20?

Jawab : [tex]2 = 1 \times 2, 6 = 2 \times 3, 12 = 3 \times 4, ...[/tex]

[tex]U_n = n(n+1)\\U_{20} = 20(20+1) = 20 \times 21 = 420[/tex]

Jadi, suku ke 20 dari pola bilangan 2, 6, 12, 20, ... adalah 420

3. Dik : 2, 4, 6, 8, ...

Dit : Suku ke 20?

Barisan aritmatika => barisan yang memiliki jarak beda (suku kedua - suku pertama) yang konstan.Barisan geometri => barisan yang memiliki rasio (suku kedua dibagi suku pertama) yang konstanBarisan fibonacci => barisan yang terbentuk dari penjumlahan dua pola sebelumnya.

1) Berdasarkan lampiran, berapa pola ke-10 ?

Diketahui : 6,9,12

Ditanya : U10 ?

Dijawab :

Un = a + (n-1) b

U10 = a + 9b

U10 = 6 + 9(3)

U10 = 6 + 27

U10 = 33

2) Pernyataan untuk mendapat bilangan kedua?

Diketahui : (3,6)(6,15)(8,21)

Ditanya : Pernyataan ?

Dijawab :

3 ke 6 = 3 >> (3 x 3 - 3)

6 ke 15 = 9 >> (6 x 3 - 3)

8 ke 21 = 13 >> ( 8 x 3 - 3)

Pernyataan : Dikali tiga dan dikurang tiga.

3) Tentukan 3 bilangan selanjutnya dari 10,30,50,70

Diketahui : 10,30,50,70

Ditanya : 3 bilangan ?

Dijawab :

beda = U2 - U1

        = 30 - 10

        = 20

70 + 20 = 90

90 + 20 = 110

110 + 20 = 130

Hasil akhir : 90,110,130

4) 3 bilangan selanjutnya dari 2,3,8,11,16

Diketahui : 2,3,8,11,16

Ditanya : 3 bilangan selanjutnya ?

Dijawab :

Angka dua kita anggap tidak ada, jadinya 3,8,11,16

beda (I) = U2 - U1

        = 8 - 3

        = 5

beda (II) = U3 - U2

             = 11 - 8

             = 3

3 bilangan selanjutnya adalah

16 + 3 = 19

19 + 5 = 24

24 + 3 = 27

[19,24,27]
